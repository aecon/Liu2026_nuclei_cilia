# Ilastik classifiers


Four pixel classifiers were trained in ilastik:

1. 3D Classifier 1 was trained on 3D COCS images of the `NBH-HPI-4` and `RML6-HPI-4` groups.  
2. 3D Classifier 2 was trained on 3D COCS images of the `NBH-CentrinoneB` and `NBH-DMSO` groups.  
3. 3D Classifier 3 was trained on 3D COCS images of the `RML6-CentrinoneB and `RML6-DMSO` groups.  
4. 2D Classifier was trained on 2D nuclei images from 96 well plates.
